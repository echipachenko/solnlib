splunksolutionlib.platform.platform.make_splunkhome_path(parts)[source]¶

create a path string by the several parts of the path. For example, parts is [“etc”, “apps”, “Splunk_TA_aws”], the return path will be $SPLUNK_HOME/etc/apps/Splunk_TA_aws. Note this function assumed SPLUNK_HOME is in environment varialbes :parts: list or tuple which consists of the path part :return: full path if succeful, otherwise throw ValueError exception
